🚀 Ejemplo de Uso - Sistema de Propuestas Dinámicas

💡 Este archivo muestra cómo usar el sistema de propuestas dinámicas de Cloud Pixels

📋 Paso 1: Crear una Propuesta

1. Abre el Panel de Administración

2. Ve a la pestaña "Crear Nueva Propuesta"

3. Llena el formulario con la información del cliente

4. Haz clic en "Generar Propuesta HTML"

5. La propuesta se guardará automáticamente con un código único

🔗 Paso 2: Compartir la Propuesta

1. Ve a la pestaña "Lista de Propuestas"

2. Encuentra tu propuesta en la lista

3. Haz clic en "🔗 Copiar Enlace"

4. El enlace se copiará al portapapeles

5. Comparte este enlace con tu cliente

📄 Paso 3: El Cliente Ve la Propuesta

Cuando el cliente abra el enlace, verá:

  • ✅ Propuesta personalizada con su información
  • ✅ Servicios específicos seleccionados
  • ✅ Entregables personalizados
  • ✅ Precio y condiciones de pago
  • ✅ Información de contacto actualizada

🎯 Ejemplo de Enlace de Propuesta

http://localhost:8000/cloud_pixels_proposal_ui_v3.html?propuesta=CP-20241201-1234

🔧 Funcionalidades del Sistema

✅ Lo que SÍ hace el sistema:

  • Genera códigos únicos automáticamente
  • Guarda propuestas en base de datos local
  • Carga propuestas dinámicamente desde la URL
  • Permite editar propuestas existentes
  • Genera enlaces compartibles
  • Descarga archivos HTML estáticos

❌ Lo que NO hace el sistema:

  • No requiere servidor web (solo para desarrollo)
  • No envía datos a servidores externos
  • No requiere base de datos compleja
  • No necesita configuración de servidor

🚀 Prueba el Sistema

Para probar el sistema completo:

📊 Ir al Panel de Administración 📄 Ver Template de Propuesta
💡 Consejo: Asegúrate de tener un servidor local corriendo (python -m http.server 8000) para que todas las funcionalidades trabajen correctamente.
